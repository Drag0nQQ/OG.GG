## Getting Started

### Installation
1. Clone the repo
```sh
git clone https://github.com/NaimanMpl/react-tournamentgg.git
```
2. Install composer packages
```sh
composer install
```
3. Install npm packages
```sh
cd chatserver
npm i
```
4. Create .env file at public/ and fill credentials below
```sh
cd public
touch .env
RIOT_API_KEY=
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_PORT=3306
MAIL_HOST=
MAIL_USERNAME=
MAIL_PASSWORD=
MAIL_REDIRECT=
```
